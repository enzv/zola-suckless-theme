Code blocks need to be obvious, predictable, and quiet.
If a theme makes code look clever, it is already doing too much.

```sh
zola build
zola check
```

```toml
[extra]
primary_nav = [{ text = "posts", href = "/posts/" }]
```

Inline code should sit inside the sentence and leave it alone.

This theme treats code as content, not costume.
