# Suckless

Zola theme. Plain on purpose.

It takes its cues from `suckless.org`: text first, no JavaScript, no webfonts, no decorative nonsense.
If you want effects, gradients, and little UI tricks begging for attention, use something else.

## What It Does

- one base template for pages, sections, taxonomies, and 404
- `primary_nav` plus optional `secondary_nav`
- section sidebar and post archive navigation
- one local stylesheet at `/style.css`
- no JavaScript
- no remote assets
- a few shortcodes, nothing bloated

## Run It

```sh
zola serve
```

## Build It

```sh
zola build
```

## Check It

```sh
zola check
```
